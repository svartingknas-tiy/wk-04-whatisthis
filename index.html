this

  1. called with new operator - new binding
    • ex: var dog = new Animal();
    • .this is the newly constructed object; e.g. 'dog'
  2. .call or .apply - hard binding
    • ex 1: whatIsThis.call(trickyTricky, 'nice', 'job');
    • ex 2: whatIsThis.apply(confusing, ['nice', 'job']);
    • .this is the explicitely specified object; e.g. 'trickyTricky' or 'confusing'
  3. function called with a context - implicit binding
    • ex: inAnObject.test1('face', 'book');
    • .this is the context object; e.g. 'inAnObject'
  4. otherwise - default binding
    • ex: var cookie = cuddle();
